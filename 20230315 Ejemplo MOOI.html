Ejemplo de la metodología MOOI¶

El MOOI es un proceso creado por el Observatorio de Desarrollo Econmico de Bogot (ODEB) para optimizar las convocatorias de la oferta institucional de la Secretara Distrital de Desarrollo Econmico (SDDE) y facilitar su evaluacin. Para esto, el MOOI utiliza una base nica de unidades productivas de Bogot y estima un puntaje que organiza las unidades productivas de acuerdo con la posibilidad que estas tienen de beneficiarse del programa. Primero, se filtra la base de datos para dejar solamente las unidades productivas elegibles para el programa. Luego, se realiza un anlisis de factores para datos mixtos, una variante de la metodologa de componentes principales, para reducir la dimensionalidad de los criterios de priorizacin de las unidades productivas participantes. Posteriormente, agrupa las unidades productivas de acuerdo a esta caracterizacin simplificada usando la tcnica k-means y, por ltimo, genera un puntaje de priorizacin de acuerdo a la distancia de la unidad productiva al centroide del grupo preferido para recibir el programa.

El programa con el que se realizar el anlisis es Camino a la inclusin financiera, el cual es parte de la estrategia de la SDDE con Bancoldex para que las unidades productivas accedan a crditos. En concreto, es un curso de formacin en finanzas para los micronegocios el cual, una vez completado, le da a los participantes un certificado con el que pueden ir a las entidades financieras y acceder a los beneficios de Bancoldex.

Nota aclaratoria: El ejemplo aqu presentado est hecho con fines indicativos y no para ser puesto en operacin. Especialmente porque, si bien lso criterios de inclusin son acertados, los criterios de priorizacin an no han sido validados con la direccin implementadora.

Selección de variables de exclusión y priorización¶

Se toman los criterios de los programas, se filtra la base y se dejan las variables de priorizacin en la forma deseada

Existen dos tipos de criterios a tener en cuenta: a. Los de exclusin: las caractersticas que necesita tener una unidad productiva para acceder al programa b. Los de priorizacin: aquellas caractersticas que, de acuerdo a la teora de cambio, hacen que una unidad productiva elegible saque mayor provecho del programa.

Para Camino a la inclusin financiera se tienen: a. Criterios de exclusin: ser un micronegocio (tener un mximo de 9 personas trabajando). b. Criterios de priorizacin: no llevar una contabilidad (porque pueden sacar mayor provecho de una formacin en finanzas), llevar ms tiempo funcionando (son negocios ms slidos y es ms factible que puedan sobrellevar un prstamo), bajos ingresos (como proxy de productividad), expresar inters en un curso de finanzas, expresar deseos de tener un crdito y ser propiedad de mujeres (los negocios de mujeres tienen menor acceso a crdito).

Las variables de priorizacin aqu elegidas sirven para ejemplificar la operacin del MOOI, sin embargo, hay muchas otras variables que se podran considerar como valiosas en este anlisis. Por ejemplo, el sector al que pertenece, la razn para comenzar el negocio y con qu entidades ha tenido o tiene prstamos, las cuales se encuentran en la Tropa Econmica y pueden ser incluidas en el anlisis, pero se dejaron por fuera para facilitar la lectura.

Imputación de datos utilizando MICE¶

Algunos registros pueden tener valores faltantes, sea por negacin al contestar por parte del encuestado o por errores de registros del encuestador. Dado lo anterior, y ante la incoveniencia de eliminar registros con algunas variables faltantes para no dejar por fuera a unidades productivas de los programas, se opta por recurrir a estrategias de imputacin.

Para la imputacin de variables continuas se utiliza el mtodo de Imputaciones Mltiples por Ecuaciones Encadenadas, conocido como MICE por sus siglas en ingls. Este es un mtodo de imputacin iterativo el cual imputa valores faltantes para cada columna utilizando una regresin bayesiana lineal y como variables independientes todas las dems columnas. Este proceso lo realiza manera iterativa hasta minimizar la diferencia en valores de una iteracin a otra.

A continuacin se define los criterios del modelo. 1. Como ecuacin para estimacin se utiliza una regresin bayesiana lineal 2. En la primera iteracin se utiliza la media de las variables para la regresin 3. Se imputa primero tomando las variables con menor cantidad de valores faltantes como variable dependiente.

Reducción de dimensiones utilizando FAMD¶

Para reducir la dimensionalidad tanto de variables continuas como categricas se utiliza el anlisis de factores para datos mixtos (FAMD en ingls). Al igual que el PCA, este anlisis parte del principio de que existen unos componentes que condensan la varianza de mltiples variables. A diferencia del PCA, el FAMD tiene en cuenta la varianza tanto de variables continuas como categricas.

            engine='auto',

Clusterización utilizando K-means¶

K-means es una de las metodologas ms usadas para hallar clsteres o cmulos de datos y su uso es tan extendido porque esos grupos tienen la caracterstica de que sus miembros son lo ms parecido posible entre s. El algoritmo comienza con una agrupacin aleatoria de las observaciones en un nmero de agrupaciones preestablecido y minimiza las distancias de cada observacin a su centroide, mientras maximiza la distancia entre los centroides. El algoritmo recursivamente reagrupa las observaciones hasta que encuentra los centroides y grupos ptimos (centroides), as, una observacin cualquiera pertenece a un grupo porque est ms cerca del centroide que lo representa.

Ejecución del proceso¶

Entonces, en el MOOI se realizan los siguientes pasos:

a) Filtros de inclusin: seleccin de las variables segn los criterios de participacin del programa. b) Caracterizacin de unidades productivas: reduccin de dimensiones de la base resultante (FAMD): este NO ES el puntaje final c) Agrupacin de unidades productivas segn caractersticas: clusterizacin de las dimensiones (k-means y elbow) d) Eleccin del perfil ms adecuado entre los grupos: ojo de experto, de acuerdo a variables ms importantes e) Asignacin de puntaje a cada potencial beneficiario: distancia al centroide: este ES el puntaje final
